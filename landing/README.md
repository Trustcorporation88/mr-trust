## 🎓 Success Tips

### ✅ DO THIS
- **Be specific**: "Economizar 30min por lead" not generic "automation"
- **Show ROI**: Every pitch includes number (money/time saved)
- **Tell stories**: Use case studies, not features
- **Follow up**: 80% of deals close after 3-5 touches
- **Track everything**: Emails sent, opens, clicks, demos, closes
- **Ask for referrals**: Happy customers are best source
- **Celebrate wins**: First customer = big deal (document it!)

### ❌ DON'T DO THIS
- Don't wait for perfect: Ship now, optimize later
- Don't blast everyone: Segment by industry/pain point
- Don't give up early: Most close on touch 3-5
- Don't forget CRM: Track EVERYTHING
- Don't ignore feedback: Use it to improve
- Don't neglect follow-up: Schedule next touch immediately
