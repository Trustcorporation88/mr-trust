# Workspace Split

Este repositório está organizado em dois projetos independentes:

- `OSINT/` -> Projeto Mr.Holmes OSINT
- `CRM/` -> Projeto CRM (app Streamlit + backend/webhook + testes)

## Execução rápida

### OSINT

```bash
cd OSINT
./.venv/Scripts/python.exe web_app.py
```

### CRM

```bash
cd CRM
python crm_app.py
```

## Publicação no GitHub

Você pode commitar/publicar cada projeto por pasta, mantendo histórico separado por contexto.
